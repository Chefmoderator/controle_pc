# **ControlFlow — система дистанційного керування ПК**

**ControlFlow** — це незалежна та безпечна система дистанційного керування комп’ютерами x86/x64 через мобільний застосунок.  
Проєкт об’єднує **ПК → власний сервер користувача → мобільний пристрій**, забезпечуючи повний контроль над системою, процесами та файлами без використання хмарних сервісів

---

## 🚀 **Основна ідея**
ControlFlow створений для того, щоб надати користувачеві **повну автономію**, можливість керувати своєю технікою **з будь-якої точки світу**, зберігаючи **анонімність**, **прозорість** і **безпеку**.

Користувач сам розгортає сервер, сам обирає, які ПК підключати, і повністю контролює процеси взаємодії. Жодні дані не збираються, не зберігаються та не передаються третім сторонам.

---

## 📡 **Архітектура системи**
[Мобільний застосунок] ⇄ [Сервер користувача] ⇄ [ПК-клієнт]

---

### **Мобільний застосунок (Flutter/Dart):**
- Інтерфейс взаємодії  
- Передача команд на сервер  
- Отримання та візуалізація відповідей ПК  

### **Сервер (Python + FastAPI):**
- Авторизація та маршрутизація команд  
- З'єднання WebSocket/http з ПК  
- Передача відповідей у мобільний застосунок  
- Логи сервера (повна прозорість для користувача)  

### **ПК-клієнт (Python):**
- Виконує команди системи  
- Повертає дані (процеси, файли, стан системи)  
- Підтримує управління гучністю, яскравістю, файлами, програмами тощо

## 🔧 **Можливості ControlFlow**

### 🖥 **Системний контроль**
- Перегляд системної інформації  
- Моніторинг температури, заряду, апаратних ресурсів  
- Перегляд uptime, користувачів, ОС  

### ⚡ **Керування живленням**
- Вимкнення  
- Перезавантаження  
- Сон  
- Гібернація  

### 🔊 **Управління аудіо**
- Mute / Unmute  
- Зміна гучності  
- Читання поточної гучності  

### 💡 **Керування яскравістю**
- Отримати рівень яскравості  
- Встановити потрібне значення  

### 📂 **Файлова система**
- Перегляд вмісту директорій  
- Пошук файлів  
- Створення файлів/папок/архівів  
- Видалення, редагування, читання  
- Отримання структури файлової системи  

### ⚙️ **Процеси**
- Перегляд усіх процесів  
- Пошук  
- Kill / Restart  
- Отримання детальної інформації  
- Запуск процесів

## 🔥 **Чим ControlFlow відрізняється від існуючих рішень?**

### **✔ Повна автономія**  
Запуск свого сервреа.  
Ніхто не має доступу до даних, окрім користувача.  

### **✔ Без хмар і корпорацій**  
На відміну від TeamViewer/AnyDesk — жодного централізованого сервісу.  

### **✔ Анонімність**  
Сервер не зберігає логінів, IP, телеметрії.  
ПК не може бути відстежено навіть технічно.  

### **✔ Мінімальні вимоги**  
Працює навіть на повільних мережах.  

### **✔ Гнучкість**  
Можна додавати власний сервер.

## 🛠 **Запуск проєкту**

### 1️⃣ Запуск сервера
```bash
cd backend_server
pip install -r requirements.txt
python main.py
```

### 2️⃣ Запуск клієнта на ПК
```bash
cd pc_app
pip install -r requirements.txt
python main.py
```
### 3️⃣ Запуск мобільного застосунку
```bash
cd android_app
flutter pub get
flutter run
```

## 🔐 Безпека

- Кожен ПК має унікальний ключ API.  
- Сервер перевіряє валідність усіх команд.  
- Сервер не зберігає персональних даних.  
- Комунікація WebSocket — ізольована та захищена.  
- Немає телеметрії, трекінгу або прихованих логів.
