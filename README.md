**ControlFlow - Cross-Device Remote Control System**

**ControlFlow** — это модульная система удалённого управления, позволяющая контролировать устройства х86-х64 (ПК, ноутбуки, серверы) с мобильного Android-приложения.

Проект передаёт команды от смартфона серверу - ПК-клиенту и обратно, обеспечивая удалённое управление и сбор системной информации.
